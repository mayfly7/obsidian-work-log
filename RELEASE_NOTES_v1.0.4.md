## 新增功能

- **待办功能**：新增"添加待办"按钮，在光标处插入可勾选待办项；日历底部常驻显示所有未完成待办，点击可跳转到对应行
- **日期格式可配置**：支持 YYYY-MM-DD / MM-DD / M月D日 / YYYY年MM月DD日 四种格式
- **一键迁移**：一键批量将旧格式日期标题转换为新格式
- **编辑器光标同步**：光标移动到日期标题行时，日历自动跳转并选中对应日期
- **日期生成模式**：支持「每天新增到当前日」（默认）和「每年自动生成全年」两种模式，切换时自动补全/裁剪
- **待办圆点标记**：有未完成待办的日期显示红色圆点 + 数量
