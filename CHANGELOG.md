# 更新日志 / Changelog

## [1.0.4] - 2026-06-11

### 新增功能

- **待办功能**：新增"添加待办"按钮，在光标处插入可勾选的待办项 `- [ ]`
- **待办面板**：日历底部常驻显示所有未完成的待办列表，点击可跳转到对应文件的行
- **日期格式可配置**：设置中支持选择日期标题格式（YYYY-MM-DD、MM-DD、M月D日、YYYY年MM月DD日）
- **一键迁移**：设置中新增"应用新日期格式"按钮，批量将旧格式日期标题转换为新格式
- **编辑器光标同步**：编辑器光标移动到某日期标题行时，日历自动跳转并选中对应日期
- **日期生成模式**：设置中可选择日期生成模式
  - `每天新增到当前日`（默认）：只生成到今天，保持文件简洁
  - `每年自动生成全年`：一次性生成全年日期结构
  - 模式切换时自动处理：全年→每天会裁剪未来日期；每天→全年会补全到年底
- **日期圆点标记**：有工作记录的日期下方显示圆点标记
- **待办圆点标记**：有未完成待办的日期显示红色圆点标记（显示待办数量）

### 优化改进

- **交互改进**：去掉右键菜单，改为左键单击日期打开对应日期内容
- **添加记录交互**：点击"添加工作记录"按钮后弹出选项，选择"上午"或"下午"
- **Hover 预览**：鼠标悬停日期时显示临时选中框，底部按钮文字动态变化
- **添加记录去重**：点击"上午"/"下午"时先检查是否已存在，已存在则光标跳到该行末尾，不存在则正常插入
- **日历位置**：移除设置项，硬编码为右侧边栏显示
- **日期格式兼容**：改日期格式后，旧格式日期标题仍能正常解析（多格式回退解析）
- **滚动定位优化**：改进滚动定位逻辑，多次重试确保大文件编辑器就绪

### 修复问题

- 修复 Win 端日历只显示"1"的问题（`this.isMobile()` → `this.app.isMobile`）
- 修复待办不可点选的问题（确保 `- [ ]` 在行首）
- 修复日期格式变更后光标同步失效的问题（改用多格式解析）
- 修复一键迁移时丢失冒号后内容的问题（正则只替换日期+星期部分）
- 修复模式切换时补全/裁剪不完整的问题
- 修复全年→每天模式切换后周标题残留的问题
- 修复"今日"按钮未清除其他日期选中状态的问题

## [1.0.0] - 2026-06-09

### 首次发布

- **年度工作日志**：支持以年度单文件形式记录每日工作（如 `2026.md`）
- **自动日期结构**：支持自动生成年度日期结构，可选择生成全年或只生成到今天
- **按月分组**：日期按月份分组，结构清晰
- **按周归类**：同一周的日期归类在一起，便于查看
- **日历视图**：提供日历视图，快速跳转至指定日期
- **自定义路径**：支持自定义工作日志文件存放路径
- **Weekly 标题**：自动添加周标题（如 `### Weekly 2026-W01`），便于按周查看
